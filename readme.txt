This plugin shows archives in a categorized way that is archives are categorized under category name and date.

== Description ==

Archives are categorized under category name and date. It filters archives based on the category and clicked date when they are being displayed in archives page. Archives page will only show posts from the clicked date and also the posts that belong to that category. So, all in all there will be two filters while viewing archives page.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'archives-category-date'
3. Activate Archives by Category and Date plugin from your Plugins page. 
4. A menu will be added at the admin panel called 'Archives by Category and Date'. Click on it & you will see the desired plugin result.

= From WordPress.org =

1. Download archives-category-date.
2. Upload the 'archives-category-date' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Archives by Category and Date' from your Plugins page. 
4. A widget called 'Archives by Category and Date' will be added in the Widgets section. Drag it to any of your sidebar and you will see the desired plugin result.

== Frequently Asked Questions ==

= How does it work? =

After activation, a widget called 'Archives by Category and Date' will be added in the Widgets section. Drag it to any of your sidebar and user will see the desired plugin result. If any date link is clicked then it will redirect user to archives page where filtered posts will be seen.
